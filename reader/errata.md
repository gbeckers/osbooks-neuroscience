# Errata — corrections to the original OpenStax text

Plain-English notes for each correction made to the **neuroscience upstream**
modules (root `modules/`). The exact byte-level record is in git
(`git diff upstream/main -- modules/`); this file adds the human explanation that
will appear in the reader's provenance appendix.

Keep errata as in-place edits committed with a `errata:` prefix, e.g.
`errata: m00012 "recieve" -> "receive"`, so the changelog is greppable:
`git log --oneline --grep '^errata:'`.
